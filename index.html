About The Project

We are Jack, Spencer, Yang, and Lin and we are CSE students. Jack is a Cognitive Science student so he LOVES to integrate neuroscience, AI, computer science, psychology, etc.

What inspired this project was when he learned about "Memory Champions" in his cognitive psychology class.

The quote we used on the home page is actually advice from a memory champion himself. This is a photo of Nelson Dellis at a memory championship. It is this concept of visual imagery that sparked the idea to turn this process of generating mental imagery from a manual process, to an automatic one.

It was then, that we realized that these techniques dont only have to be used for memory championships, they can also be used to help you exceed academically. Your memory is malleable, it can get worse, or it can get better, and there is verifiable evidence that mental visual imagery is the key that unlocks the great extent of your memory :]
